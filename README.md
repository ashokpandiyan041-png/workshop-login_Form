# Responsive Workshop Registration Form

A responsive workshop registration form built using **HTML5** and **CSS3**. The project features a clean two-column layout for desktop screens and automatically adapts to tablets and mobile devices using CSS Flexbox and Media Queries.

## 📸 Preview

A modern and user-friendly registration form for a Web Development Workshop.

## 🚀 Features

* Responsive design for Desktop, Tablet, and Mobile
* Clean and modern user interface
* Built with HTML5 and CSS3
* Flexbox-based layout
* CSS Media Queries for responsiveness
* Font Awesome icons
* Input fields with validation support
* Submit and Reset buttons

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Font Awesome
